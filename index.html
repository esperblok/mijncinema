⚙️ Instellingen

De site werkt al meteen — er is een sleutel ingebouwd. ✅
Wil je je eigen gratis TMDB-sleutel gebruiken? Vul 'm hieronder in (optioneel).

Hoe kom ik gratis aan een sleutel?
  1. Ga naar themoviedb.org/signup en maak een gratis account.
  2. Open Instellingen → API.
  3. Klik op "Create" / "Request an API key" → kies Developer.
  4. Vul het formulier in (bij website mag je bv. http://localhost invullen).
  5. Kopieer de "API Key (v3 auth)" en plak die hierboven. Klaar! ✅
